**Purpose of this folder and document**:

- There were some changes made throughout minor version revisions that necessitated migration scripts to avoid breaking existing beta users. When 3.0 is planned for a release, the folder and this file should be reviewed since the major version bump would clearly indicate that it includes app-breaking changes.

- **remove-filter-rules-migration**: on 3/24/26, I changed the settings named "filters" to "rules" to avoid confusion with the viewing filters. To avoid breaking existing beta users' rules, I included a migration script that would convert the old filters to the new rules. When 3.0 eventually gets released, this should be removed. Files most likely able to be removed: src\utils\settings-migration.ts, test_files\unit\keyword-rules-migration.test.ts, and 'migrateKeywordRulesSettings' keyword. (marcd35)
