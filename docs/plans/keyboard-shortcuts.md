# Global hotkeys

- Settings/Preferences — `Alt + p`
- Open this help — `h`
- Close dialog — `Esc`

# Dashboard View

- Unread articles — `Shift + 1`
- All articles — `Shift + 2`
- Refresh feed — `r`
- List view — `1`
- Card view — `2`
- Feed view — `3`

- Focus previous article — `p`
- Card view navigation — `← → ↑ ↓`
- Open/Close article — `o / Enter`

- Load mobilized content — `w`
- Mark as read/unread — `m`

- Like — `l`
- Broadcast — `s`
