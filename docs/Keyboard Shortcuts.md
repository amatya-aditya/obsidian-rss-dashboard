# Keyboard Shortcuts

## General Navigation

| Shortcut | Action                         |
| -------- | ------------------------------ |
| ?        | Open Help Dialog               |
| Esc      | Close Dialog / Clear Selection |
| r        | Refresh Feed                   |

## Dashboard View

| Shortcut  | Action                 |
| --------- | ---------------------- |
| Shift + s | Focus sidebar          |
| Shift + r | Focus reader view      |
| Shift + 1 | All articles filter    |
| Shift + 2 | Unread articles filter |
| Shift + 3 | Read articles filter   |
| 1         | List view              |
| 2         | Card view              |
| 3         | Feed view              |

## Reader View

| Shortcut               | Action                       |
| ---------------------- | ---------------------------- |
| Shift + d              | Focus dashboard view         |
| Shift + s              | Focus sidebar                |
| Shift + r              | Focus reader view            |
| ArrowUp / ArrowDown    | Scroll article up/down       |
| ArrowLeft / ArrowRight | Scroll article left/right    |
| PageUp / PageDown      | Scroll by one page           |
| Home / End             | Jump to start/end of article |
| = / +                  | Increase font size           |
| - / \_                 | Decrease font size           |
| 0                      | Reset font size              |

## Article Manipulation

| Shortcut        | Action                                |
| --------------- | ----------------------------------- |
| Arrow keys      | Card view navigation                |
| o / Enter       | Open article in reader pane         |
| k               | Close reader pane                   |
| j               | Open prior article in feed          |
| l               | Open next article in feed           |
| m               | Mark article read/unread toggle     |
| ,               | Mark article read and open next     |
| Shift + a       | Mark all as read                    |
| f               | Star/Unstar article                 |
| t               | Add tags to article                 |
| s               | Save full content to notes          |

## Sidebar Navigation

| Shortcut                  | Action               |
| ------------------------- | -------------------- |
| Shift + l                 | Next item            |
| Shift + j                 | Previous item        |
| ArrowUp / ArrowDown       | Move focused item    |
| ArrowLeft / ArrowRight    | Jump between folders |
| Shift + o / Shift + Enter | Open focused item    |
| Shift + x                 | Open/Collapse folder |
| Shift + d                 | Delete folder/feed   |
| Shift + r                 | Rename folder/feed   |
