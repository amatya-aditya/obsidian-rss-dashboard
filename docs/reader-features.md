# Reader Menu Features

The Reader View provides a highly customizable environment for consuming your RSS feeds, YouTube descriptions, and more. Access the **Reader Settings** via the format icon in the header to fine-tune your experience.

## Layout & Precision

- **3-Row Organization**: Each setting is cleanly separated with its name, description, and control on individual rows for maximum clarity.
- **Hybrid Controls**: All sliders are now paired with numeric input boxes. You can drag for quick changes or type exact values for pixel-perfect precision.
- **Instant Sync**: Changes apply immediately to your reading view as you adjust them.

## Available Settings

### Typography & Spacing

- **Font Scale**: Scale text from 80% to 200%. Perfect for adapting to different screen sizes.
- **Line Height**: Adjust vertical spacing (120% to 220%) to improve scanning speed and readability.
- **Font Family**:
  - **Theme Default**: Matches your Obsidian workspace.
  - **Serif**: Ideal for long-form, literary content.
  - **Sans**: Clean, modern, and highly legible.
  - **Mono**: Best for technical content and code snippets.
- **Paragraph Spacing**: Choose between **Tight**, **Normal**, or **Loose** gaps between blocks of text.

### Layout & Alignment

- **Text Alignment**:
  - **Justify**: Clean, magazine-style edges on both sides.
  - **Left**: Standard left-aligned text for a natural reading flow.
- **Words Per Row**: Control the line length.
  - _Tip: Set to `0` for full-width reading, or a specific value (like `15-20`) for a focused "reading column"._

### Management

- **One-Click Reset**: Quickly revert all reading preferences to their default state.
